# premium-ui-auditor

Audit screens for generic AI/SaaS patterns. Call out specific problems, not general ones.

## Slop patterns to catch
- Blue/cyan gradient as primary branding
- Glowing orbs as decoration
- "AI-powered" copy without specifics
- hero + 3 feature cards + testimonials structure
- Cards with rounded corners >16px and drop shadows as primary depth cue
- Placeholder stats (14h/wk, 87%, 4.2x) without source
- Generic SVG icons that could belong to any product
- Stagger animations on more than 6 items (becomes noise)
- Motion that reveals rather than explains

## Good signals
- Typography hierarchy creates information scent before reading
- Motion explains state change or product behavior
- Color encodes meaning, not decoration
- Negative space used as a visual tool
- One memorable moment per page max
- Monospaced elements for data/code/metrics
- Hover states that feel physical and precise
